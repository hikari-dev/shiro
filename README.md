# shiro

1. Create a config file named **shiroConfig.yml** under the same folder where your jar file at.

```yml
bot:
  qq: 666666
  password: password

database:
  url: jdbc:mysql://123.123.123.123:1234/dbname
  driverClassName: com.mysql.cj.jdbc.Driver
  username: username
  password: password

masterQQ: 123456
```

2. Run shiro.jar
